# Morfx v1.0.0

The baseline Morfx release introduces the deterministic AST transformation
engine and its MCP server shape.

## Highlights

- Deterministic AST-based code transformations instead of regex edits
- MCP server for AI coding clients
- Query, replace, delete, insert, append, and staged apply primitives
- Multi-language support through tree-sitter providers

## Scope

This reissued `v1.0.0` is kept as the historical baseline release in the new
public release timeline.
